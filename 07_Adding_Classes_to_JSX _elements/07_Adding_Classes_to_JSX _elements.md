### ** الدرس العاشر من سلسلة ريأكت جي إس (React JS)**  

#### **أهداف الدرس:**  
- التعرف على كيفية إضافة الـ **className** في JSX.  
- فهم الفرق بين `class` في HTML و `className` في JSX.  
- معرفة كيفية كتابة الأكواد الخاصة بالتصميم في ملفات CSS وربطها مع المكونات.  
- تعلم كيفية استخدام `className` بطريقة ديناميكية باستخدام الجافا سكريبت.

---

#### **محتوى الدرس:**  
1. **مراجعة الدرس السابق:**  
   - تعلمنا كيفية إضافة خاصية `style` في JSX.  
   - استخدمنا الأقواس `{}` لإسناد كائن `object` يحتوي على خصائص CSS.  

2. **إضافة الـ `className` في JSX:**  
   - في HTML، نستخدم `class="class-name"`، لكن في JSX يجب استخدام `className="class-name"`.  
   - نضع القيمة بين `{}` إذا كانت متغيرة أو تعبير جافا سكريبت.  

3. **التعامل مع ملفات CSS الخارجية:**  
   - نكتب الأكواد في ملف `.css` ونستوردها داخل ملف الجافا سكريبت باستخدام `import './style.css';`.  
   - نستهدف العناصر باستخدام محددات CSS مثل `.active { border: 10px solid green; }`.  

4. **إضافة `className` بشكل ديناميكي:**  
   - يمكننا استخدام الجافا سكريبت داخل JSX لإضافة `className` بناءً على شروط معينة:  
     ```jsx
     const isActive = true;
     <h1 className={isActive ? "red-bg" : "blue-bg"}>Hello</h1>;
     ```
   - هنا، إذا كان `isActive` يساوي `true` فسيتم تعيين `className="red-bg"`، وإذا كان `false` سيكون `"blue-bg"`.  

5. **استخدام عدة `className` معًا:**  
   - نفصل بين أسماء الكلاسات بمسافة داخل النص:  
     ```jsx
     <h3 className="active red-bg">Welcome</h3>
     ```

6. **استخدام الجافا سكريبت لإنشاء `className` ديناميكي مع المتغيرات:**  
   - إذا كنا نريد تحديد الكلاس بناءً على قيمة معينة:  
     ```jsx
     const userName = "yacoub";
     <h1 className={userName === "yacoub" ? "red-bg" : "blue-bg"}>Hello</h1>;
     ```
